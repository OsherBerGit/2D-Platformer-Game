# 2D Platformer Game

A 2D platformer game developed using Godot Engine 4.5.

This project was created by following the Coco Code tutorial.

## Features

- 2D Platforming mechanics
- GDScript Scripting
- Mobile platform support

## Controls

- **Move Left**: Arrow Left / A
- **Move Right**: Arrow Right / D
- **Jump**: Space / Arrow Up / W
- **Pause**: Esc

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/OsherBerGit/2D-Platformer-Game.git
   ```

2. Open the project in Godot Engine 4.5 or later.

3. Run the main scene.
